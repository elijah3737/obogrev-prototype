# SPEC — пиксельный эталон obogrev.ru (для прототипа)

Замерено с живого сайта при вьюпорте **1440px**. Это ground truth для аудита.

## Палитра
- Синий (кнопки/каталог/в корзину): `#0058FF`
- Жёлтый (найти/акции/заявка): `#FFD700`
- Красная цена / бейджи: `#F0392A` / `#ED1C24`
- Зелёный (скидка/наличие): `#3DB54A`; жёлтая «экономия» pill
- Оранжевый (о компании, спец-пилюли, CTA): `#F58220`
- Тёмно-синий заголовки категорий: `#16306B`
- Текст: `#1a1d24`; вторичный `#4d4d4d`/`#5a6172`; светлый фон секций/плиток `#f5f5f7`
- Шрифт: **CeraPro** (платный) → свободный аналог **Manrope**

## Шапка (3 ряда, sticky)
- **Топбар**: высота ~34px. Слева ссылки (О нас, Блог, Отзывы, Купить оптом, Доставка, Оплата, Контакты) — 14px / 700 / `#4d4d4d`. Справа 3 телефона (700) + email (синий).
- **Mid-ряд**: высота ~80px. Логотип (бокс ~180×41: красный круглый знак + «Brand logo» + подпись «ваш поставщик тепла с 2010 г»).
  - **Каталог**: высота **51px**, padding **17px / 33px**, font **17px / 500**, radius **10px**, bg `#0058FF`, иконка-сетка слева.
  - **Поиск**: input высота **48px**, font 14px, обёртка radius **16px**; кнопка **найти** жёлтая, чёрный текст, та же высота, font 16.
  - **Иконки** (Сравнение, Корзина, Избранное, ЛК): svg **24px**, подпись ~12px под иконкой, синий бейдж-счётчик сверху-справа.
- **Nav-ряд**: высота ~39px. Ссылки 14px / 700, вертикальный паддинг ~6px, с «▾». Справа жёлтая pill **«% Акции»** (14px).

ВАЖНО: кнопки компактные — НЕ раздувать паддинги.

## Главная (порядок секций)
1. Топбар → 2. Шапка → 3. Nav
4. **Промо-слайдер**: контентная ширина (≈ как `.wrap`), высота **~420px**, скруглён (~16–20px), красный градиент, крупный белый заголовок-капс, точки снизу по центру.
5. **Плитки категорий** 2-в-ряд: bg `#f5f5f7`, radius **20px**, высота ~347px. Пастельная круглая иконка, навигационный навиго-синий заголовок ~24px, серое описание, «Подробнее →» (белая pill с синей рамкой), картинка товара справа-снизу.
6. Жёлтый баннер «Ещё больше товаров… жми!» + иконка-рука.
7. 3 преимущества (иконка слева + заголовок + текст).
8. **Табы товаров** (Хит продаж / Рекомендуем / Новинки / Распродажа) + сетка карточек (5 в ряд на десктопе).
9. Блок «Работаем на рынке… с 2010 года» (оранжевый круг-«солнце» + картинка дома; справа текст + колонки Статус/Услуги/Партнёрская сеть с оранжевыми подзаголовками + «Подробнее о нас»).
10. Блог (3 карточки: картинка/дата/заголовок/«Читать далее»).
11. CTA «Остались вопросы?» (оранжевый заголовок + форма ФИО/Телефон/синяя кнопка; справа картинка с оранжевыми пилюлями).
12. Футер (4 колонки: Каталог / Услуги / Информация / соцсети+рейтинг+OZON) + копирайт.

## Карточка товара (в сетке)
Бейдж top-left (Хит продаж/Распродажа красный, применение синий, Новинка зелёный) · избранное + сравнение top-right (круглые) · медиа: кольцо-водяной знак + бренд + оранжевые спец-пилюли + точки · название 16px · «N отзывов» (синий) + жёлтые звёзды · разделитель · «Цена за метр:» · **красная цена** + старая зачёркнутая · зелёный «−N%» + жёлтая «экономия ₽» · наличие (точка) · «Купить в 1 клик» / «Купить оптом» (подчёркнутые) · степпер «− 1 +» + синяя pill «В корзину».
**Клик по медиа/названию → переход на СТРАНИЦУ товара (НЕ поп-ап).**

## Страница товара (#/product/<id>) — отдельная страница
1. Хлебные крошки.
2. **H1** 28px / 800 / чёрный.
3. Ряд действий: Сравнить · В избранное · Задать вопрос · Поделиться (слева) + «% Хочу скидку!» (справа, синий).
4. **3 колонки**:
   - **Галерея** (слева): колонка миниатюр (4 шт + prev/next) + большое изображение с кольцом-водяным знаком, бренд-лого, бейджи (синие), спец-пилюли (оранжевые круги SRL / Мощность 24 Вт/м).
   - **Характеристики** (центр): бренд-лого + название бренда; «Модель» — кнопки-варианты (активная синяя рамка); спец-строки с **точечными лидерами** (Мощность … 24, Назначение … …, Монтаж … Наружный, Толщина … 4,5, Страна … Южная Корея); «Все характеристики >».
   - **Блок покупки** (справа, **sticky**, белый, radius 20, padding 20, ~280px): красная цена + «за м. с НДС» + «Цена без учёта скидки», «В наличии» (зелёная точка), степпер «− 1 +», синяя pill «В корзину» (во всю ширину), синяя-outline pill «Купить оптом», жёлтая «Оставить заявку на почту», «Информация о доставке» (Самовывоз: сегодня / Курьером: бесплатно, завтра / Пункты выдачи — с синими подссылками).
5. Ряд доверия: Продукция сертифицирована · Профессиональная консультация · Оперативная доставка.
6. Описание + таблица характеристик (вкладки/блок).
7. «С этим берут» / похожие товары (сетка карточек).

## Бренд-имя
Везде вместо «ЭТК Обогрев» — плейсхолдер **«Brand logo»**.
