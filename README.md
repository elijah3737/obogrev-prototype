# obogrev-prototype — прототип интернет-магазина «Brand logo»

Кликабельный прототип нового сайта. Главная страница и карточки товара сделаны
**один в один по референсу obogrev.ru** (синий/жёлтый/красный, шрифт CeraPro→Manrope).
Название бренда — плейсхолдер «Brand logo». **Без калькуляторов.**

## Запуск
Статика, без сборки. Превью-сервер: `obogrev-prototype` (порт 8802) в `.claude/launch.json`,
либо вручную: `python3 -m http.server 8802` из этой папки → http://localhost:8802

## Структура
- `index.html` — оболочка (топбар, шапка, поиск, мегаменю, корзина, модалка, тосты)
- `css/tokens.css` — дизайн-токены (тема «Heat»: тёплая бумага, жар-оранжевый акцент, стальной cool для B2B)
- `css/styles.css` — компоненты, адаптив (320/375/678/1080+)
- `js/data.js` — каталог-движок: 13 категорий, подкатегории, фасеты, **по 8 тестовых карточек в каждом разделе** (104 товара), готовые комплекты, кейсы, услуги
- `js/app.js` — рендер, hash-роутинг, фасетные фильтры, корзина, 1-клик, КП, формы

## Что внутри (реализованные улучшения)
- **Навигация по задаче** — плитки «Кровля/Трубы/Площадки/Пол/Резервуары/Протечки»
- **Проф-фильтры** — бренд, применение, тип, Вт/м, взрывозащита, темп.класс, площадь, покрытие, функции, цена (реально фильтруют)
- **Готовые комплекты** — решение целиком, не деталь
- **Богатый сниппет товара** — цена + ₽/м + Вт·м + наличие + срок поставки + «В корзину» + «1 клик» + «Запросить КП» + гарантия + «Нашли дешевле?»
- **B2B-оффер** — топбар, отдельный блок, КП с карточки
- **Услуги** — этапы, гарантия, бесплатный замер (лид-форма)
- **Кейсы с цифрами**, триггеры доверия (дистрибьютор, доставка, гарантия)

## Важно
Прототип фронтенда. Бэкенда нет — формы и оформление заказа показывают тост-заглушку.
Карточки товаров — тестовые; в проде приедут по API-синхронизации с заводами.
Шрифты — Manrope + Golos Text (Cyrillic, вне «чёрного списка» DESIGN.md).
